Definitions Types

Definities Definiëren

Definities in Semantic zijn groepen van css, fonts, afbeeldingen, en javascript die samen een enkel element vormen. Anders dan bij andere javascript bibliotheken, zijn Semantic UI elementen stand-alone en hebben alleen hun eigen assets nodig om correct te functioneren.

Definitie Types

Semantic heeft vijf verschillende ui definities. Deze zijn bruikbare patronen voor het beschrijven van herbruikbare delen van een website.

UI Element Een basis bouw blok van een website, bestaat alleen of in homogene groepen.
UI Collectie Een heterogene groep van verschillende elementen welke gewoonlijk gezamelijk aangetroffen worden.
UI View Een wijze om algemene website inhoud zoals comments of activiteit feeds te presenteren.
UI Module Een element voor wie z'n gedrag een essentieel onderdeel is van z'n definitie.
UI Behavior Een set op zichzelf staande acties die niet aan een specifiek element zijn gekoppeld.

Type Definities

UI Elementen

UI Elementen zijn pagina elementen welke ondeelbaar zijn. Dit kan worden gezien als vergelijkbaar zijnde aan een "element" in de scheikunde.

UI Elementen kunnen meervoudige definities hebben wanneer zij bekend staan als samen te bestaan binnen groepen. Meervoudige variaties staan je toe om kwaliteiten slechts eenmalig te specificeren, zodat zij afgeleid kunenn worden over alle leden van een groep heen.

UI Collecties

UI Collecties zijn groepen van heterogene pagina elementen welke gewoonlijk gezamenlijk gebruikt worden. De scheikundige metafoor voort zettend, kan men deze zien als moleculen.

UI collecties hebben een definitie van elementen die in hen bestaan, of in hen zouden moeten bestaan. Ze vereisen meestal niet dat alle elementen gevonden kunnen worden, maar ze beschrijven een lijst men de "gebruikelijke verdachten". Anders dan element definities, zijn collecties meestal niet nuttig om meervoudig te definiëren.

Voorbeelden van UI collecties:

  • Formulieren
  • Tabellen
  • Grids (Layout)
  • Menus

UI Modules

UI Modules zijn elementen wiens gedrag een fundamenteel onderdeel zijn van z'n definities. UI Modules zijn meestal afhankelijk van javascript welke de implementatie van z'n gedrag bevat, alhoewel het gedrag in sommige speciale omstandigheden allen css kan bevatten.

Voorbeelden van UI modules:

  • Popups
  • Modals
  • Chat Rooms

UI Views

UI Views zijn veel voorkomende stijlfiguren voor de presentatie van specifieke soorten informatie. Anders dan collecties welke zich specifiek op de relatie tussen interface elementen focussen, focussen views zich specifiek op de relatie van site inhoud met andere stukken site inhoud.

Voorbeelden van UI views:

  • Comment Feed
  • Activiteit Feed
  • Produkt Lijst
