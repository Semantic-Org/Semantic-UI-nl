Types

Een type is een wederzijds exclusieve versie van een element

Overzicht

Overzicht

Een ui definitie in Semantic bevat gewoonlijk een lijst van wederzijds exclusieve variaties op een element z'n ontwerp. Een type wordt aangewezen door een additionele class name toe te voegen aan een UI element

Inhoud

Types vereisen mogelijk verschillende HTML structuren om correct te werken. Een icoon menu kan bijvoorbeeld andere inhoud verwachten zoals een icon glyph in plaats van tekst om correct weergegeven te worden

HTML Verschillen

Types vereisen elk mogelijk ook licht verschillende HTML. Een gestapeld menu heeft bijvoorbeeld HTML voor een submenu nodig om zichzelf correct weer te geven.
